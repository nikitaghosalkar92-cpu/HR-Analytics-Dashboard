# HR-Analytics-Dashboard
Visualizes key HR metrics such as employee count, attrition rate, tenure, age, training, and promotions.

Shows attrition insights by job role, department, marital status, education field, and age groups.

Includes salary hike trends and average monthly income comparison across roles.

Provides detailed job satisfaction rating analysis based on different roles.

Contains interactive charts for gender-wise attrition, employee age distribution, and department-wise exits.

Helps HR teams understand attrition patterns and workforce performance for better decision-making.
